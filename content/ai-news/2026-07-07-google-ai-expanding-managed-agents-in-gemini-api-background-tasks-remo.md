## What was announced

Google AI has expanded the Gemini API's Managed Agents with new features such as background task execution, remote MCP server connection, custom function support, and credential refresh.

Developers can now build more reliable and production-ready agents with these new capabilities. This compares favorably to other AI APIs that lack these features. Developers should immediately start exploring and integrating these new functionalities into their projects.

- Background task execution allows for asynchronous interactions, improving agent responsiveness.
- Easier connection to remote MCP servers simplifies agent deployment and management.
- Custom functions and credential refresh enhance agent flexibility and security.
