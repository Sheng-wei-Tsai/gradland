## What was announced

OpenAI launched the Campus Network, a global program supporting student clubs with access to OpenAI tools, API credits, event hosting resources, and educational materials. The initiative aims to help university student organizations build AI-powered projects and communities locally. It provides structured support infrastructure for clubs worldwide to run workshops, hackathons, and collaborative AI learning.
