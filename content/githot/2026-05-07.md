**How to use it**

1. Install: npm install @struktoai/mirage-node
2. Instantiate and mount sources:
   `const \{ MirageFS \} = require('@struktoai/mirage-node');`
   `const fs = new MirageFS();`
   `await fs.mount('s3', new S3Adapter(\{ bucket: 'my-bucket', region: 'ap-southeast-2' \}));`
   `await fs.mount('gdrive', new GoogleDriveAdapter(\{ credentials \}));`
