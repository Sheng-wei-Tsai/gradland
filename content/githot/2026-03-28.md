**935 stars this week** · TypeScript

A TypeScript SDK that bridges any AI agent backend (OpenAI, Claude, Codex) into WeChat's messaging interface via a minimal chat() interface and QR code login.

**Use case**

If you want to expose an AI assistant through WeChat — China's dominant messaging app — without reverse-engineering its API or dealing with official bot approval bureaucracy, this SDK handles the auth, message polling, and media decryption for you. Concrete example: you build a Claude-powered coding assistant and want Chinese colleagues to query it via WeChat instead of a separate web app — three lines of code and a QR scan gets you there.

**Why it's trending**

The ACP (Agent Client Protocol) open standard just launched, and this repo is one of the first real-world implementations showing how ACP-compatible agents (Claude Code, Codex, kimi-cli) can be piped into a mainstream consumer platform with zero glue code — making it a reference architecture for the 'any agent, any surface' pattern everyone is talking about right now.

**How to use it**

1. Install the SDK: `npm install weixin-agent-sdk`
2. Implement the Agent interface with your AI backend — for OpenAI it's a straightforward messages array accumulation inside chat():
```typescript
import OpenAI from 'openai';
import { login, start, type Agent } from 'weixin-agent-sdk';

const client = new OpenAI();
const histories = new Map<string, OpenAI.ChatCompletionMessageParam[]>();

const agent: Agent = {
  async chat({ conversationId, text }) {
    const history = histories.get(conversationId) ?? [];
    history.push({ role: 'user', content: text });
    const res = await client.chat.completions.create({ model: 'gpt-4o', messages: history });
    const reply = res.choices[0].message;
    history.push(reply);
    histories.set(conversationId, history);
    return { text: reply.content ?? '' };
  }
};

await login();  // prints QR code to terminal — scan with WeChat
await start(agent);
```
3. Run the script, scan the QR code with your personal WeChat account.
4. Send a WeChat message to yourself or a contact — the agent responds automatically.
5. For existing ACP agents (Claude Code), skip custom code entirely: `npx weixin-acp claude-code`
