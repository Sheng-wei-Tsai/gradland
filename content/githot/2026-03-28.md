**935 stars this week** · TypeScript

A TypeScript SDK that bridges any AI agent backend (OpenAI, Claude, Codex, etc.) into WeChat's messaging interface using a dead-simple Agent interface.

**Use case**

Developers with existing AI backends have no clean way to expose them through WeChat without reverse-engineering Tencent's private APIs. This SDK abstracts the WeChat Clawbot protocol so you implement a single `chat()` method and get a fully functional WeChat AI assistant — for example, wiring Claude Code to respond to WeChat messages from your phone without touching any WeChat internals yourself.

**Why it's trending**

WeChat's new Clawbot feature just opened up AI assistant integration, and ACP (Agent Client Protocol) is gaining traction as a cross-agent standard this week — this repo is the first clean TypeScript bridge between the two ecosystems, landing at the right moment when developers are actively experimenting with both.

**How to use it**

1. Install the SDK: `npm install weixin-agent-sdk`
2. Implement the Agent interface with your AI backend logic (see snippet below)
3. Call `await login()` — this triggers a QR code scan in your terminal to authenticate your WeChat account
4. Call `await start(agent)` to begin the message loop
5. For ACP-compatible agents like Claude Code, skip all of that and just run `npx weixin-acp claude-code`

```typescript
import { login, start, type Agent, type ChatRequest } from 'weixin-agent-sdk';
import OpenAI from 'openai';

const client = new OpenAI();
const histories = new Map<string, Array<{role: string, content: string}>>();

const myAgent: Agent = {
  async chat(req: ChatRequest) {
    const history = histories.get(req.conversationId) ?? [];
    history.push({ role: 'user', content: req.text });
    const res = await client.chat.completions.create({
      model: 'gpt-4o',
      messages: history as any,
    });
    const reply = res.choices[0].message.content ?? '';
    history.push({ role: 'assistant', content: reply });
    histories.set(req.conversationId, history);
    return { text: reply };
  },
};

await login();
await start(myAgent);
```
